# Mobile Agent Skills

Concrete skill folders live in `mobile/.agent/skills/`. Use this file as the human-readable index; use the `SKILL.md` files as the executable agent guidance.

## mobile-product-architect

Use for app map, module parity, release phasing, and product scope decisions. This skill owns whether a mobile feature belongs to Worker, Union Admin, shared, or future PUWF/Government scope.

## mobile-legal-forms-mapper

Use before creating forms or fields. This skill maps official PUWF legal/governance documents to mobile form fields, validation, labels, and document evidence requirements.

## mobile-urdu-rtl-ux

Use for Urdu-first flows, low-literacy worker copy, RTL/LTR behavior, language switching, and typography QA.

## mobile-design-system

Use for design tokens, PUWF/FOS branding, icon usage, visual hierarchy, dense mobile layout, and avoiding generic template UI.

## mobile-expo-engineer

Use for Expo Router, NativeWind, Reanimated, TanStack Query, Zustand, SecureStore, notifications, file/document APIs, and mock-service architecture.

## mobile-qa-accessibility

Use for device QA, RTL testing, form validation testing, accessibility labels, loading/empty/error states, performance, and release-readiness checks.
