# PUWF Mobile Master Plan

## Product Vision

The PUWF mobile app is the worker and union-admin companion to the web platform. It gives factory workers, union members, and union office bearers a fast Urdu-first way to verify membership, file grievances, manage dues, vote, view rights, and operate union workflows from a phone.

## Target Users

- Worker: low-to-moderate digital literacy, Urdu-first, needs direct access to digital ID, dues, rights, grievances, voting, and union contact details.
- Union Admin: office bearer or data operator who needs dense mobile access to members, office bearers, grievances, dues, elections, documents, annual returns, and CBA/CoD status.
- Public helper: unauthenticated user who needs role selection, guidance, and grievance/help entry.

## App Map

- Splash: PUWF logo, FOS attribution, theme preload.
- Role Selection: worker, union admin, public help, language switch.
- Worker Onboarding: survey-style first-run flow.
- Worker Portal: dashboard, digital ID, dues, grievances, voting, rights, my union, notifications.
- Union Admin Portal: dashboard, members, office bearers, dues/finance, annual return, elections, CBA/CoD, grievances/legal, documents/compliance.
- Shared: language, profile, settings, notification center, empty/error/loading states.

## Module Priority

1. Shell, localization, theme, mock data.
2. Worker onboarding and worker dashboard.
3. Worker grievances, digital ID, dues, voting, rights.
4. Union admin dashboard and member registry.
5. Union admin operations: annual returns, elections, CBA/CoD, grievances/legal, documents/compliance.
6. QA hardening, device testing, accessibility, RTL polish.

## Phase Roadmap

- Phase 0: Documentation foundation and mobile agent rules.
- Phase 1: Expo foundation, app routing, theme, localization, mock services, shared UI.
- Phase 2: Splash, role selection, login mocks, worker onboarding, route guards.
- Phase 3: Worker portal modules.
- Phase 4: Union admin core modules.
- Phase 5: Union admin operations modules.
- Phase 6: polish, QA, accessibility, performance.
