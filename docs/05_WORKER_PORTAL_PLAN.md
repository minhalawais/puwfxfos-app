# Worker Portal Plan

## Screens

- Dashboard: identity strip, dues, grievance, benefits, vote, rights, and latest notice.
- Digital ID: PUWF card, masked CNIC, QR placeholder, offline availability state.
- Dues: monthly ledger, current status, outstanding months, receipt detail.
- Grievances: guided filing, voice-note placeholder, attachments placeholder, SLA timeline, resolution confirmation.
- Voting: active election, candidate cards, OTP mock, cast confirmation.
- Rights: Urdu-first rights library with categories for wages, safety, union rights, EOBI/social security, termination.
- My Union: union details, office bearers, CBA status, documents, contact actions.
- Notifications: unread notices, grievance updates, dues alerts, election reminders.

## UX Rules

- Each screen must show primary status in the first viewport.
- Use compact rows and chips; avoid tall empty hero panels.
- Worker-facing Urdu must stay simple and reassuring.
- All modules must support empty states for first-time workers.

## Acceptance Criteria

- Worker can complete onboarding and land on dashboard.
- Worker can navigate all bottom-tab modules in Urdu and English.
- Worker can create a mock grievance and see tracking information.
- Worker can view digital ID and dues history from mock data.
